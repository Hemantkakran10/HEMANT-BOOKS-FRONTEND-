# 📚 Hemant Find — Find Books You Want!

Hemant Find is a clean and minimal book discovery web application
designed to help users find, explore, and stay updated on books
and curated content. Built with a cozy library aesthetic that makes
every visit feel like walking into your favourite bookstore.

## ✨ Features
- 🔍 Book search functionality — find books instantly
- 📖 E-Book Corner — curated digital book collection
- 📢 Notice Board — stay updated with new announcements
- 🆕 New Update section — latest additions and changes
- 📬 Contact section — get in touch easily
- 🎨 Beautiful bookshelf-themed UI with warm aesthetics
- ⚡ Fast, lightweight — pure HTML, CSS & JavaScript

## 🛠️ Tech Stack
- HTML5
- CSS3
- JavaScript (Vanilla)

## 🚀 Getting Started
git clone https://github.com/yourusername/hemant-find
cd hemant-find
open webpage/index.html
